# Hnefatafl

## Introduction

Ce projet a été réalisé en BUT1 lors d'une SAE (Situation d'apprentissage évaluée) seul, dans laquelle nous devions recréer le jeu de plateau Hnefatafl. Pour contextualiser, Hnefatafl est un jeu de plateau viking asymétrique avec un but simple : un roi au centre doit s’échapper vers un bord ou un coin, et en face, les attaquants tentent de le capturer.
Les pièces se déplacent verticalement et horizontalement et capturent en encerclant une pièce sur deux côtés.


## Travail effectué

Pour cette SAE, 2 livrables était attendus, le premier était l'affichage simple du tableau et le deuxième, un jeu fonctionnel. De plus, le jeu développé dévait passer les 13 tests créés par les professeurs.

J'ai donc reproduis le jeu Hnefatafl dans le terminal de l'IDE (QTCreator) en C++ avec un affichage dans le terminal avec les tests passés.
